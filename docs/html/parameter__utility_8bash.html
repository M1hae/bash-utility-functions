◆ RegisterParameter()

RegisterParameter ( )

Register parameters.

Return values
Errorcode0 input fits the restrictions
Errorcode1 input is something else
Parameters
[in]StringParameter (e.g -i)
[in]StringBash command or function (e.g. echo)
[in]StringHelp text
[in]String(Optional) Category default: None

Register input parameter in the following pattern:
RegisterParameter <Parameter> <Function/Command> <Help text> <Category>
E.g.:
RegisterParameter "-i" "echo" "Wrapper for echo"
RegisterParameter "-i" "echo" "Wrapper for echo" "General"

NOTE:

  • You cannot register -h, -help, –h, –help because it is used for the help screen
  • You need to sort the parameters yourself
